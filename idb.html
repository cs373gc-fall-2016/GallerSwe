Artist(DB.Model)
The artist model contains attributes about an artist such as  name and birth date. Artists have a number of Artworks.  To get the styles of an artist, filter the ARTWORK_STYLE table with all of their artworks.
       
Artwork(DB.Model)
The artwork model contains attributes about an artwork such as name, medium, description, and style
       
Collection(DB.Model)
The collection model contains attributes about where a piece is located, the website of the institution, and the type of institution
       
Style(DB.Model)
The style model contains attributes about a style such as name, description, and  time period
